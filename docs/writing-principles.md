# Writing Principles

> Language guidelines for all B8E website copy.
> Applied by hp-designer and enforced by hp-guardian.

---

## The Voice

B8E writes the way a grounded person speaks quietly to someone they trust.

- Not an expert explaining to a student
- Not a salesperson closing a deal
- Not a coach motivating a client

More like: a person who has walked a long path, and is telling you what they found there.

---

## Sentence Structure

**Short sentences. One thought at a time.**

```
✅ 外側を変えても、内側が動かなければ、本当の変容は起きない。
❌ 外側を変えることは重要ですが、内側の変容も同時に行うことで、真の意味での変革が実現できます。
```

**Line breaks are breathing room, not decoration.**
Break at the point where the reader needs a pause — not at grammatical limits.

**Questions open space. Declarations close it.**
```
✅ 自分が本当に望んでいることが、わからなくなってきた。
❌ 自分が本当に望んでいることを、もう一度見つけましょう。
```

## Tone Words

### Use freely
```
在る / 存在する   — being, existing
問い             — question (not problem)
流れ             — flow (natural state)
整える           — to align, to bring into order
伴走             — walking alongside
気づく           — to notice, to become aware
深まる           — to deepen (not improve)
戻っていく        — to return to (not to progress toward)
```

### Use sparingly or avoid
```
支援します         — "we support you" (feels service-adjacent)
提供価値          — "provided value" (corporate)
ソリューション     — solution (jargon)
最適化            — optimization (tech/business)
まずはお気軽に     — "feel free to consult" (formulaic)
ご相談ください     — "please consult us" (formulaic)
実現できます       — "we can realize" (promise language)
当てはまる        — "applies to / fits into" (categorizing)
```

## Specific Patterns

### Hero copy
- 1–2 lines maximum
- Should raise a question or reframe an assumption
- Never announces the page topic

```
✅ 変容には、外側と内側がある。
✅ ツールの問題ではない、と気づいたところから始まる。
✅ 自分の内側に、戻っていく場所。
❌ DXコンサルティングで、あなたの会社を変革します。
```

### Section headings (`.section-block h3`)
- Rendered as small uppercase labels via CSS
- Should name the perspective, not summarize the content

```
✅ 伴走という在り方
✅ Dreamin' Engine
✅ 螺旋的成長とは
❌ 提供するサービスの内容
❌ こんな方にお勧めです
```

### Resonance questions (`.resonance-questions`)
- 3–5 short statements, not questions — the form is declarative, but they land as questions
- Each one mirrors a real inner state the reader might be in
- No answer is given. The space after each line is the answer.

```
✅ 目標はある。でも、なぜか動けない。
✅ 変わりたい。でも、何が邪魔しているかわからない。
❌ 目標はあるのに動けない方へ、私たちがサポートします。
```

### CTA copy
- One paragraph maximum (2 sentences)
- One link, text-only — no buttons, no emphasis
- The link text should be an action the reader chooses, not one they're urged into

```
✅ 「準備は整っていなくていい。問いがあれば、それで十分です。」
   → 話してみたいと思ったら

✅ 「まだ決めていなくていい。まず、話すところから始められます。」
   → 話してみたいと思ったら

❌ 「今すぐお問い合わせください！無料相談受付中。」
   → [お問い合わせはこちら]
```

## What to Avoid

| Pattern | Why |
|---|---|
| Feature lists (箇条書きの機能羅列) | Creates a shopping experience, not resonance |
| "こんな方に向いています" | Categorizes the reader as a target |
| Testimonials or social proof | Signals that persuasion is the goal |
| Numbered steps or processes | Implies the reader must follow instructions |
| Bold text for emphasis | The writing should carry its own weight |
| Multiple CTAs on one page | Creates urgency through repetition |

---

## The Final Test

Read the copy out loud.
If it sounds like a brochure, revise it.
If it sounds like a quiet conversation, it's ready.
